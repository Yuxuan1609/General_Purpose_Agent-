# 对局分析技能

## 流程
1. 逐局审阅执行记录，关注输掉的局和接近的局
2. 识别每局的决策链条：玩家手牌 → 层链推理 → 最终动作
3. 对比成功局的策略和失败局的策略差异
4. 提炼可复用的模式改进建议

## 输出格式
- 失败模式：[手牌/局面] → [错误推理] → [错误动作] → [建议修正]
- 成功模式：[手牌/局面] → [正确推理] → [正确动作] → [值得保留]
- L2 卡片建议：需要新增/修改/降级的知识卡片
- L1 规则建议：需要新增或修正的行为准则

## 原则
- 基于统计证据而非单次运气偏差
- 修正建议需保持泛化能力，不针对特定对局过拟合
- 区分运气失败和策略失败的差异
